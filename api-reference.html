Events

AUCTION_COMPLETE

The auction has completed

AUCTION_GO

Start the publisher auction

AUCTION_LIB_LOADED

Auction provider library loaded

AUCTION_LIB_START

Auction provider library load started

AUCTION_POST_RUN

The auction has finished running

AUCTION_REFRESH

The auction was restarted

AUCTION_TRIGGER

Start the publisher auction from a business rule. e.g. a bidder timeout

BID_ASSEMBLER

Start bid assembler

BID_COMPLETE

Action is complete

BID_LIB_LOADED

Bid provider library loaded

BID_LIB_START

Bid provider library load started

BID_PUSH_NEXT

Next item in data stream ready.
e.g BidProvider.refresh raises a BID_PUSH_NEXT event for each bid.

BID_START

Action started.
e.g BidProvider.init

ERROR

Error event raised

INVALID

Invalid operation or data event raise

PUBFOOD_API_LOAD

Api library load

PUBFOOD_API_START

Api library start

WARN

Warn event raised
